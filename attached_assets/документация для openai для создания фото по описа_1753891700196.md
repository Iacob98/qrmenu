# документация для openai для создания фото по описанию

Для создания фото по описанию с помощью OpenAI используется модель DALL·E и соответствующий API. Основная официальная документация и инструкции доступны как на русском, так и на английском языках.

**Краткая инструкция:**

- Для генерации изображения применяется эндпоинт POST https://api.openai.com/v1/images/generations.
- В теле запроса указываются параметры:
    - **model** — название модели, например, "dall-e-3"
    - **prompt** — текстовое описание изображения (максимальная длина зависит от версии модели)
    - **n** — количество изображений (для dall-e-3 всегда 1)
    - **size** — разрешение: “1024x1024” и другие варианты (для dall-e-3 также доступны “1792x1024” и “1024x1792”)
    - **response_format** — формат ответа: “url” или “b64_json”[^2].

**Пример запроса cURL**:

```bash
curl https://api.openai.com/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $OPENAI_API_KEY" \
  -d '{
    "model": "dall-e-3",
    "prompt": "A cute baby sea otter",
    "n": 1,
    "size": "1024x1024"
  }'
```

В ответе вы получите ссылку на сгенерированное изображение в поле "url"[^2].

**Особенности:**

- Для DALL·E API доступны функции генерации, редактирования и создания вариаций изображений (редактирование и вариации чаще работают с DALL·E 2)[^2].
- Для стабильной работы необходим актуальный API-ключ от OpenAI.
- У изображений обычно есть ограниченное время хранения по ссылке — сохраняйте их, если необходимо[^1].

**Документация (на русском):**

- Основные сведения, параметры, форматы и примеры официально изложены здесь: https://openai-docs.ru/docs/api-reference[^2].
- Описание самих моделей: https://openai-docs.ru/docs/models[^6].

**Быстрый старт на Python:**

- Используйте официальную библиотеку openai:

```python
import openai

openai.api_key = "<ваш_ключ>"

response = openai.Image.create(
    model="dall-e-3",
    prompt="A cute baby sea otter",
    n=1,
    size="1024x1024"
)
url = response["data"][^0]["url"]
```

- Более подробный разбор с примерами кода: https://proglib.io/p/iskusstvennoe-voobrazhenie-sozdanie-izobrazheniy-iz-teksta-s-pomoshchyu-dall-e-2-i-openai-api-2023-02-07[^1].

**Важные параметры запроса:**


| Параметр | Описание | Пример |
| :-- | :-- | :-- |
| model | Модель DALL·E (“dall-e-2”, “dall-e-3”) | “dall-e-3” |
| prompt | Описание изображения | “кошка в очках” |
| n | Кол-во изображений (1–10, для DALL·E 3 — только 1) | 1 |
| size | Размеры (“256x256”, “512x512”, “1024x1024”, др.) | “1024x1024” |
| response_format | “url” или “b64_json” | “url” |

Если требуется примеры редактирования изображений или описания других функций API, уточните ваш запрос.
