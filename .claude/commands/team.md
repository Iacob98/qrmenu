# Team — Запуск всей команды QRMenu

Ты — **оркестратор команды**. При вызове этой команды ты запускаешь всех специалистов параллельно через Agent tool для комплексного анализа задачи.

## Команда

| Роль | Команда | Зона ответственности |
|------|---------|---------------------|
| Тимлид | `/team-lead` | Координация, приоритеты, распределение |
| Архитектор | `/architect` | Проектирование, архитектура, API дизайн |
| DB-инженер | `/db-engineer` | Схемы, миграции, запросы, оптимизация |
| Backend | `/backend` | API, сервисы, middleware, интеграции |
| Frontend | `/frontend` | React UI, компоненты, состояние |
| QA | `/qa` | Тестирование, баги, безопасность |

## Инструкции

При вызове `/team`:
1. Запусти 3-4 наиболее релевантных специалиста параллельно через Agent tool
2. Каждый агент должен проанализировать задачу со своей стороны
3. После получения всех результатов — составь сводный отчёт как Тимлид
4. Предложи конкретный план действий

Если `$ARGUMENTS` пустой — запусти team-lead для анализа текущего состояния проекта.

Задача от пользователя: $ARGUMENTS
